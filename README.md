1.项目名称：试题库管理系统
2.数据库名称：tests_manage_system
3.工程搭建环境：jdk1.8+ecilpse+mysql server 5.7+tomcat 8.5

4.数据库搭建：
(1)、打开mysql
(2)、新建数据库tests_manage_system
(3)、在数据库中执行tests_manage_system.sql文件（在工程文件中）

5.项目启动步骤：
(1)、打开eclipse
(2)、在eclipse中打开工程
(3)、新建tomcat服务器
(4)、将项目部署在服务器上
(5)、修改数据库的账号密码（在db.properties文件中修改）
(6)、启动项目

6.默认网址：localhost:8080/tst_mana_sys/login.action
